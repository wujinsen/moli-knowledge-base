## 结论

词话本中的 **方外人物** 不是背景装饰，而是西门庆 **盛极—自毁—散尽** 的三段功能镜：

| 轴 | 代表 | 功能 | 关键回 |
|----|------|------|--------|
| **命** | [[吴神仙]] | 子平/麻衣相，早定加官与血光、呕血流脓 | 29、30、61、79 |
| **欲** | [[胡僧]] | 春药赠予，纵欲催命，「死于极乐」 | 49–52、79 |
| **利** | [[薛姑子]] · [[王和尚]] · [[大师父]] | 争法事主顾、擗分经钱 | 62–73 |
| **因果** | [[普静师]] | 永福寺度孝哥，西门庆转世偿债 | 89、100 |

**命—欲对照** 是主轴：第29回吴神仙已言西门庆「癸亥运发」后「呕血流脓之灾」，月娘却疑「相随心灭」；第49回同一极盛窗口又遇胡僧赠药——**预言与春药同回并置**（屈体求荣 / 遇胡僧施药），盛极与杀机并生。第79回吴神仙再诊「膏肓难疗」，与胡僧药过量暴亡 **前后印证**，把「色」之反噬收束为肉身注脚。

> 词话本第49回 **仅一胡僧**，无「叶番僧」专名；旧误建页已删，见 [[胡僧]] 页说明。

### 时间线（西门庆）

```
第29回 吴神仙冰鉴 ──→ 预言加官 / 六六之年呕血流脓
第30回              ──→ 「平地登云」应验
第49回 胡僧赠药   ──→ 纵欲物质起点（与蔡御史盐引同回）
第61回              ──→ 月娘忆李瓶儿「三九血光」，吴神仙已往武当
第73回 僧尼争利   ──→ 薛姑子斥王和尚「利心重」
第79回 吴神仙再诊 + 胡僧药反噬 ──→ 精脱血竭而亡
第100回 普静度孝哥 ──→ 冤魂荐拔，因果闭环
```

## 论据（带出处）

### 命：吴神仙

- 第29回：周守备差吴神仙至，为西门庆批八字，言「平地登云」「癸亥运发」，又断「不出六六之年，主有呕血流脓之灾」（chapters/金瓶梅/词话本/029）。
- 第29回：相李瓶儿「三九前后定见哭声」、春梅「三九定然封赠」；月娘不信珠冠之说（chapters/金瓶梅/词话本/029）。
- 第30回：西门庆云「吴神仙相我不少纱帽戴……今日果然」（chapters/金瓶梅/词话本/030）。
- 第61回：月娘忆「三九血光」，差人问周守备，回称吴神仙往武当山（chapters/金瓶梅/词话本/061）。
- 第79回：吴神仙于土地庙卦肆再诊，言「酒色过度……病在膏肓，难以治疗」（chapters/金瓶梅/词话本/079）。

### 欲：胡僧

- 第49回：蔡御史饯行后，西门庆于永福寺遇胡僧，赠百十丸春药，嘱「每次只一粒，不可多了」（chapters/金瓶梅/词话本/049）。
- 第49回：胡僧诗赞药力「一夜歇十女，其精永不伤」，西门庆欲讨药方，僧不受银、只取布而去（chapters/金瓶梅/词话本/049）。
- 第79回：潘金莲贪欢强灌，西门庆精脱血竭暴亡（chapters/金瓶梅/词话本/079；亦见 [[西门庆脱阳之症]] 与 /jinpingmei/shi 胡僧药谶）。

### 利：僧尼争主顾（辅轴）

- 第62–66回：李瓶儿丧，[[吴道官]]、黄真人、[[薛姑子]] 等轮番法事（chapters/金瓶梅/词话本/062–066）。
- 第73回：薛姑子怨 [[王和尚]]「利心又重」「擗分经钱一半方罢」（chapters/金瓶梅/词话本/073）。

### 因果：普静（后二十回收束）

- 第100回：普静荐拔冤魂，点化孝哥为西门庆转世，度出家法名明悟（chapters/金瓶梅/词话本/100；见 [后二十回散场人物](/jinpingmei/topics/后二十回散场人物)）。
