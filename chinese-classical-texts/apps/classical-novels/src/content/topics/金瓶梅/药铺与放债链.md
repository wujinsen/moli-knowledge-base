## 结论

《金瓶梅》白银叙事沿 **四条专题轨** 展开，本页与 `/jinpingmei/chain`、`/jinpingmei/silver` 交易清单互证：

```
药铺·经营 ──→ 放债·帮闲 ──→ 政商贿赂 ──→ 南货扩张 ──→ 衰败连坐
  第1–33回       第31–55回       第23–75回       第67–81回       第81–97回
```

| 轨道 | 关键节点 | 代表事件 ID |
|------|----------|-------------|
| 药铺·经营 | [[西门庆生药铺]] → 来旺酒店 → 绒线铺 | jpm-fe-001 / 009 / 004 |
| 放债·帮闲 | [[吴典恩]] · [[李智]] · [[黄四]] · [[应伯爵]] | jpm-fe-008 / 003 |
| 政商贿赂 | [[翟管家]] → [[蔡京]] 寿礼 | jpm-fe-007 / 002 / 011 |
| 衰败连坐 | 来保侵财 · 官钱粮追赃 · [[僧宝儿]] 流落 | jpm-fe-005 · 第97回 |

**核心判断**：西门庆的发家靠 **药铺本业 + 帮闲圈分利**；极盛靠 **政商庇护** 把白银转成权力；衰败则沿 **同一批借方与伙计** 反噬——吴典恩负恩、来保欺主、黄四李智钱粮连坐，构成经济叙事的闭环。

## 论据（带出处）

### 药铺·经营轨

- 第1回：西门达遗下 **五间门面生药铺**，西门庆与傅伙计算帐起家（chapters/金瓶梅/词话本/001 · jpm-fe-001）。
- 第1回：结拜十兄弟，帮闲凑分资、玉皇庙酒席（jpm-fe-006）。
- 第26回：三百两开 [[来旺]] 酒店（jpm-fe-009 · jpm-tx-002）。
- 第33回：四百五十两开绒线铺，[[韩道国]] 三七分利（jpm-fe-004 · jpm-tx-004）。

### 放债·帮闲轨

- 第31回：[[吴典恩]] 借一百两，后改「无点恩」（chapters/金瓶梅/词话本/031 · jpm-fe-008）。
- 第38–55回：[[应伯爵]] 说合，[[李智]] · [[黄四]] 借本银揽香蜡批文（chapters/金瓶梅/词话本/038 · jpm-fe-003 / jpm-tx-010）。
- 第43回：李智、黄四同来西门府关香蜡银子（chapters/金瓶梅/词话本/043）。

### 政商贿赂轨

- 第23回：认 [[翟管家]] 干亲，百两认干礼（jpm-fe-007）。
- 第27回：蔡府寿礼三百两（jpm-fe-002）。
- 第75回：二次寿礼五百两（jpm-fe-011）。

### 南货与拐财（轨间衔接）

- 第67回：韩道国、[[来保]] 四千两南货，[[崔本]] 湖州二千两（chapters/金瓶梅/词话本/067）。
- 第81回：韩拐财上东京、来保侵财欺主被逐（chapters/金瓶梅/词话本/081 · 见 [扬州经商线](/jinpingmei/topics/扬州经商线)）。

### 衰败·连坐轨

- 第95回：吴典恩负心、玳安完房（chapters/金瓶梅/词话本/095）。
- 第97回：黄四、李三（[[李智]]）与「保官儿」汤保 **官钱粮追赃**；[[僧宝儿]] 流落跟马（chapters/金瓶梅/词话本/097）。
