## 结论

红楼梦 `poems`（诗词意象）模块的本质是 **图谶式文本可视化**：把表层意象（判词/诗词/象征物/花签）经可溯源的「推论边」（`inference`）连到人物命运，再画互文网络。

迁移评估：

三部书各有一套底层架构：**西游=出世的丹道修心，金瓶=入世的因果欲念，红楼=双层宇宙的神话谶纬与宿命悲剧**。

- **红楼梦 —— 模板亦是升级母体。** 现有 `poems` 模块即源型；可进一步升级为「太虚幻境 ↔ 人间贾府」双层镜像网络（见第四节），把判词做根节点、五行金木做三角动力学。
- **西游记 —— 最该借鉴。** 韵文密度最高，且有成熟的丹道解经传统；与现有 `quanshi`（内丹心性）模块天然联动。意象内涵为「丹道符号系统」而非图谶。
- **金瓶梅 —— 独成一格。** 诗词套语偏劝诫，但「名字谶纬 + 物象隐喻」体系完整：以书名嵌名（金/瓶/梅）与酒色财气为坐标，可做「欲起→聚敛→极盛→反噬→散尽」的因果报应闭环网络。
- 三部书共用同一套 `imagery` collection、`shi.astro`、`ImageryGraph` 组件，**仅需加 `poems` feature + 建数据**，技术成本低。
- 跨书共性：红楼「大观园节点」、西游「环境节点」、金瓶「时序网络」同属**结构/环境节点**家族——某节点状态变化会牵动关联节点的命运（抄检大观园 = 关联节点清零）。

### 二、西游记借鉴方案（丹道意象网络）

不同于图谶预言，西游的意象是**内丹修炼符号**（参刘一明《西游原旨》、陈士斌《西游真诠》）：

| 意象 | 五行 | 隐喻 | 推论边指向 |
|------|------|------|-----------|
| 心猿 / 金公 | 金 | 心 / 神（金性刚锐） | 孙悟空 |
| 意马 | —（驭意） | 意 / 念 | 白龙马 |
| 木母 | 木 | 情 / 性（木性柔而易生欲） | 猪八戒 |
| 黄婆 | 土 | 调和金木（真铅真汞） | 沙僧 |
| 婴儿 | 火 / 真铅 | 难控的火性与狂躁 | 红孩儿（第41回） |
| 姹女 | 水银 / 真汞 | 消耗元阳的杂念 | 盘丝洞妖（第72回） |
| 灵台方寸山·斜月三星洞 | — | 拆字即「心」 | 修心起点（第1回） |

**五行生克机制（关系边的语义来源）**：意象之间不是静态标签，回目本身就在写丹道运作——

- **金木相克 / 交并**：悟空与八戒的冲突即「金木相克」，和好即「金木交并」。如第85回〈心猿妒木母，魔主计吞禅〉。
- **黄婆调和**：沙僧每次劝架，不只是性格设定，更是内丹「土调金木」的机制。
- **婴儿失控**：第41回〈心猿遭火败，木母被魔擒〉——红孩儿（圣婴大王）即难制的「真铅·火性」。
- **姹女耗阳**：第72回〈盘丝洞姹女求阳，木母（八戒）忘形落底〉——妖精直指消耗元阳的杂念。

**进阶 · 水火相济与环境节点（本体设计建议）**：

- 新增一组关系 subtype `水火相济 / 降龙伏虎`：悟空（火/金）与白龙马（水），在通天河、火焰山等劫难隐喻「抽坎填离 / 水火既济」。
- 引入**环境节点（Environment Node）**作为新的节点种类：把「火焰山」「通天河」等地理劫难作为节点，连到心猿/意马等人物节点，使图谱表达「特定环境 = 特定丹道工夫的考验场」。这要求 imagery schema 的 `target_kind` 除 `character` 外再支持 `location` / `environment`。

**落地约定**：

- 建议 subtype：`丹道意象` / `回前诗` / `证道偈`（如乌巢禅师《心经》）/ `地名隐喻` / `水火相济`。
- 与 `quanshi` 内丹视角互链：意象图做诠释专题的可视化底座。
- 出处可溯：心猿意马大量见于回目与回前诗（第7、14、30、41、72、85 回回目均可直引）。

### 三、金瓶梅借鉴方案（名字谶纬与世情物语网络）

不同于丹道修仙的超脱，金瓶梅的意象极度贴近肉身与世俗，以**谐音双关 + 物象隐喻**暗示宿命轨迹与家族盛衰，总纲即「酒色财气」：

| 意象 / 符号 | 隐喻 | 酒色财气 | 推论边指向 |
|------|------|----------|-----------|
| 金（潘金莲） | 锐利、杀伐、贪欲 | 色 / 财 | 西门府内斗的「刀」与毁灭引擎 |
| 瓶（李瓶儿） | 容器、聚财、易碎 | 财 / 酒 | 财富暴涨与迅速抽空 |
| 梅（庞春梅） | 严寒幸存、野蛮生长 | 气 / 运 | 倾覆后的见证者 / 上位者 |
| 花子虚 | 繁华虚妄、一场空 | — | 前期财富权力的本质（镜花水月） |
| 陈经济 | 反讽「不经济」（败家） | — | 家族防线内部溃烂，终摧西门家业 |
| 雪狮子（白猫） | 嫉妒具象、阴险杀戮 | — | 金莲恶念的物理延伸（扑杀官哥儿→大崩塌） |
| 葡萄架 | 欲望极致、庇荫与倾覆 | 色 | 纵欲巅峰场域，由盛转衰的物象坐标（第27回） |
| 胡僧药（春药） | 贪婪无度、饮鸩止渴 | 色 | 欲望反噬，盛极而亡的催命符（死于极乐） |

**建议 subtype（节点 / 关系分类）**：

- `名字谶`（人名隐喻）：吴月娘（无月 / 暗室）、宋惠莲（命贱如草）、王六儿（溜儿 / 逢迎）。
- `物象谶`（权力与欲望的信物）：「鞋」（三寸金莲，勾引私情载体）、「簪子」（妻妾地位转移争夺）。
- `曲牌 / 宝卷谶`（文本内预言）：尼姑说唱宝卷、盲女琵琶曲，常是当下命运或未来悲剧的「画外音」。

**进阶 · 因果报应闭环（本体设计建议）**：

- 构筑「欲起—聚敛—极盛—反噬—散尽」的**有向时序网络**，以西门庆为中心枢纽（Hub）。
- **转折点 = 李瓶儿之死**：物象节点（财、药、衣冠）前期**向心聚集**，转折后**离心崩解**，所有连线最终指向「空 / 死」。
- **边属性（Edge Attributes）**：引张竹坡「冷热金针」批语——表面「极热（繁华纵欲）」与底层「极冷（死亡虚无）」的张力，可作边的冷热标注。这是金瓶版的图谱本体特色（对应西游的「环境节点」）。

**落地约定**：

- 与 `quanshi` 世情视角互链：因果闭环图做诠释专题底座。
- 出处可溯：第1回开篇《酒色财气歌》定调四大意象坐标；物象流转轨迹散见各回（如李瓶儿财物如何流入西门庆、死后又如何被瓜分殆尽）。
- 注意：金瓶梅诗词多程式化套语，宜筛**有命运指向 / 物象流转**者入库，避免堆砌。

### 四、红楼梦升级方案（神话图谶与宿命意象网络 · 双层镜像）

红楼是本模块的源型（见第一节），但可再升级。其意象网络极依赖**物象绑定 + 诗词预言**：人物命运在第1、5回已于系统底层（太虚幻境）设定，人间推演只是执行该程序。

| 意象 / 符号 | 隐喻（五行/真灵） | 推论边指向 |
|------|------|-----------|
| 顽石 / 神瑛侍者 | 历劫、情根 / 玉（真） | 贾宝玉（精神内核与肉身载体） |
| 绛珠仙草 | 报恩、还泪 / 木（灵） | 林黛玉（木石前盟的神话基石） |
| 金锁 / 冷香丸 | 世俗法则、压抑 / 金（理） | 薛宝钗（金玉良缘的物质凭证与伦理约束） |
| 风月宝鉴 | 真假、色空、生死 | 贾瑞（直接触发）+ 贾府表里不一的镜相结构 |
| 水与泥 | 清净 / 污浊二元 | 群芳（水作的骨肉）vs 须眉（泥作的骨肉） |
| 大观园 | 青春乌托邦、太虚的人间投影 | 诸芳庇护所；一旦毁坏（抄检）关联节点清零 |
| 落花 / 柳絮 | 青春消亡、命运飘零 | 黛玉葬花 · 宝钗扑蝶 · 探春远嫁的情感共振 |

**建议 subtype（与现有四类对齐 / 扩展）**：

- `太虚图谶`（命运预警机制）：第5回「金陵十二钗判词」+「红楼梦十四支曲」作为**根节点（Root Nodes）**，每条判词是一条粗推论边精准指向结局（如「三春去后初雷震」→ 元春之死）。≈ 现有 `judgment`。
- `谶语载体`（日常预言介质）：诗社咏菊、元宵灯谜、酒令占花名（宝钗牡丹「任是无情也动人」、探春杏花「必得贵婿」）。≈ 现有 `flower_lot` / `poem`。
- `五行生克假说`（金木之争）：黛玉属木、宝钗属金，金（世俗婚姻）克木（自由恋爱），构成宝黛钗三角的底层动力学。

**进阶 · 双层镜像网络（本体设计建议）**：

- 图谱设上下两层：**上层太虚幻境**（警幻仙姑、薄命司、判词、神瑛侍者）/ **下层人间贾府**（大观园、十二钗、日常琐事）。imagery 节点增 `layer`（太虚 / 人间）属性。
- **映射边（Mapping Edges）**：上下层由「投胎 / 历劫」「梦境 / 警醒」连接（绛珠仙草—还泪→林黛玉垂直对齐）；下层节点最终都经一条「归彼大荒 / 千红一窟」边流回上层虚无。
- **影身对齐边**：晴雯为黛玉之影、袭人为宝钗之影——丫鬟之死预演主子结局，是一种特殊的下层内部对齐边。

**落地约定**：

- 与 `quanshi` 神话宿命视角互链：双层镜像图做诠释专题底座。
- 出处可溯：第1回（女娲补天、木石前盟、甄士隐梦幻识通灵）建神话宇宙模型；第5回（游幻境指迷十二钗）录入全部命运数据；后续以诗词、丫鬟之死、抄家逐一验证第5回判词。

### 五、适配度对比

| 维度 | 红楼梦 | 西游记 | 金瓶梅 |
|------|--------|--------|--------|
| 韵文密度 | 高 | 极高 | 高（曲多） |
| 解释传统成体系 | 是（脂评 / 探佚） | 是（丹道解经） | 较散（张竹坡） |
| 与现有模块联动 | 已 `live`（本模块源型） | `quanshi` 直接复用 | 弱 |
| 意象→命运可溯源 | 极强（判词预录） | 强 | 强（名字谶 / 物象流转，曲词弱） |
| 本体特色 | 双层镜像 + 根节点判词 | 环境节点 + 五行生克 | 时序闭环 + 冷热边属性 |
| 实现成本 | 低（升级现有） | 中 | 中 |

## 实现清单（待落地，按需启动）

1. `books/<书>.md` 的 `features` 增 `poems`。
2. `lib/books.ts` 对应书的 `shi` 模块改 `live`，更新 `desc` 与 preview 文案（西游强调丹道、金瓶强调名字谶/物象流转）。
3. 建数据 `src/content/imagery/<书>/*.md`（先覆盖核心 10–15 条）。
4. 若 subtype 超出现有四类（`judgment/poem/symbol/flower_lot`），在 imagery schema 增枚举值（西游补 `丹道意象/回前诗/证道偈/地名隐喻/水火相济`；金瓶补 `名字谶/物象谶/曲牌宝卷谶`），并在 `lib/imagery.ts` 的 `subtypeLabel` 补标签。
5. **环境节点（西游进阶）**：imagery `links[].target_kind` 增 `location` / `environment`，使「火焰山」「通天河」等可作节点连入心猿意马网络；`ImageryGraph.tsx` 按 `target_kind` 区分节点形状/配色。
6. **有向时序网络 + 边属性（金瓶进阶）**：imagery `links[]` 增 `phase`（聚敛/极盛/反噬/散尽）与 `temperature`（冷/热，对应张竹坡冷热金针）字段；`ImageryGraph` 支持按转折点（李瓶儿之死）切换向心/离心布局、按冷热着色边。
7. **双层镜像 + 映射边（红楼进阶）**：imagery 节点增 `layer`（太虚/人间）属性，`links[].kind` 增 `mapping`（投胎·还泪·梦境）与 `shadow`（影身：晴雯→黛玉）；`ImageryGraph` 上下分层布局，根节点（判词）加粗，并把「大观园」作环境节点（毁坏置灰关联节点）。
8. 可选：`<书>.imagery-links.json` 补额外互文边（西游金木相克/交并、水火既济；金瓶物象流转链；红楼影身/归彼大荒回流）。
9. 运行 `npm run build` 验证；更新对应 `index.md` 与 `log.md`。
