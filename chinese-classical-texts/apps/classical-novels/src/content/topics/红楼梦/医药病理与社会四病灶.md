## 结论

从「医药」窗口切入，《红楼梦》可见两层结构：

1. **微观**：大观园是精致的「高级监护室」——人物病症与其性格、阶层、医疗可及性高度同构；**病即性格，方即宿命**（文学—病理互文）。
2. **宏观**：监护室外的社会处于 **医疗垄断、参茸泡沫、巫医蒙昧、高夭折率** 四重压力之下；荣国府如晚期 [[王熙凤]]，表面钟鸣鼎食，内里已「油尽灯枯」（**社会学 inference**，需与原文 fact 分读）。

本站 [`/honglou/medicine/inference`](/honglou/medicine/inference) 以六轴曲线 + 社会分层可视化上述推演；本章回 [`医药诊脉链`](医药诊脉链.md) 存 **fact 时间线**，[`医药饮食名录`](医药饮食名录.md) 存 **fact / inference / literary** 三层总表。

## 二、全社会四病灶（宏观 · inference）

### 病灶一：医疗资源阶级垄断

- **原文对照**：主子层 [[王太医]]、[[张友士]] vs 丫鬟层 [[胡庸医]]、走方郎中。
- **推演**：顶级技术圈养在权力顶层；底层在庸医、重药、高昂诊费中自生自灭——**医疗不对称**是阶级矛盾的隐形导火索。

### 病灶二：人参金融化与进补泡沫

- **原文**：全书参茸温补着墨密集；第77回 [[上等人参]] 告罄、[[凤姐小月脉案]]。
- **推演**：八旗权贵「全民进补」使人参成类似奢侈品的硬通货；资源枯竭与贾府财政掏空同向（**经济史 inference**）。

### 病灶三：巫医混杂与科学蒙昧

- **原文**：[[毛半仙占卦]]、[[刘姥姥求神]]、[[赵姨娘办理后事]]（马道婆魇魔法）；巧姐惊风 [[发散风痰药]] 与 [[真牛黄]] 并见。
- **推演**：面对天花、邪病、精神异常，精英认知仍半在医学、半在神棍——公共卫生与智力上的整体蒙昧（**18世纪中西对照 inference**）。

### 病灶四：高夭折率的人口周期

- **原文死亡链**：[[贾珠]]（青年）、[[秦可卿]]、[[林黛玉]]、[[晴雯]]（16岁）、贾琏未命名子嗣等。
- **推演**：即使贾府亦「高出生、高死亡、低均寿」；底层农村婴幼儿与产妇死亡率更残酷——医药书写是 **人口学** 的写实切片。

## 三、因果链（微观 → 宏观 → 隐喻）

```text
人物病症（性格+生活方式+可及医疗）
    → 六轴分布（脉案精细 / 参茸 / 方剂 / 求神 / 庸医 / 急症）
        → 社会分层（权贵 / 中层 / 市井 / 制度性）
            → 四病灶（垄断 / 泡沫 / 蒙昧 / 夭折）
                → 隐喻：药罐子听见时代碎裂（文学—社会学 inference）
```

**终极隐喻（inference）**：荣国府如晚期凤姐——靠过期人参、[[冷香丸]] 与符咒维持体面时，「油尽灯枯」只是时间问题。此说为 **合成型主题解读**，不可当作历史定论。
