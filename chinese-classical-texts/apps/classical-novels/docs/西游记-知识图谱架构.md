# 西游记 · 知识图谱架构

## 实体层

| 集合 | 路径 | 规模（2026-06-14） |
|------|------|---------------------|
| characters | `characters/西游记/` | 55（含 `type: monster` 妖怪） |
| artifacts | `artifacts/西游记/` | 20（法宝实体） |
| chapters | `chapters/西游记/` | 100 回（世德堂本） |
| locations | `locations/西游记/` | 27（route GIS） |
| events | `events/西游记/` | 81 劫难（xy-e-001…081） |

## 功能模块（features）

| feature | 路由 | 数据依赖 |
|---------|------|----------|
| reader | `/xiyouji/read` | chapters + `西游记.chapter_summaries.json` |
| graph | `/xiyouji/graph` | `西游记.relations.json` ← `build_relations.py` |
| bestiary | `/xiyouji/bestiary` | characters where type=monster |
| nan | `/xiyouji/nan` | events subtype=tribulation |
| route | `/xiyouji/route` | locations coord/layer/route_order + events |
| items | `/xiyouji/items` | `artifacts/西游记/`（10 件） |

## 脚本

| 脚本 | 用途 |
|------|------|
| `import_chapters.py` | 世德堂本 100 回导入 |
| `seed_route.py` | 27 地点 + GIS 坐标 |
| `seed_tribulations.py` | 81 劫难 event；`EXTRA`  enrichment |
| `build_relations.py` | 人物 relations → relations.json |
| `apply_chapter_summaries.py` | JSON → 回目 summary |
| `trust_guard.py` | first_appear 正文校验 |
| `tag_chapter_meta.py` | 回目 `items[]`/`locations[]` 正文扫描 |

## 工作流

1. `/ingest`：更新回目 `characters[]`/`locations[]`/`summary`（只改 frontmatter）
2. 新建/更新人物页 `relations`
3. `python scripts/build_relations.py 西游记`
4. 劫难 enrichment：改 `seed_tribulations.py` 的 `EXTRA` → `python scripts/seed_tribulations.py`
5. 记录 `src/content/西游记.log.md`

## 与红楼梦差异

- **无双版本**：仅世德堂本 100 回，无 `variants`/`contradicts` 层（暂不适用）
- **events 层**：81 难为西游独有；红楼无 events collection
- **route GIS**：locations 含 `coord`/`layer`/`route_order`，供 `/route` 地图
