# 红楼梦 · 建筑园林学考证

## 问题域

大观园是**叙事空间**还是可复原的园林图样？本站采「文献—叙事」双轨，不强行单一平面图。

## 四组问题

1. **方位**：荣府—宁府—大观园相对位置；「入园门见障山」与后文路径
2. **布局**：正殿、省亲别墅、各姊妹院落的叙事顺序（第17–18回题额）
3. **植物**：海棠、竹、杏、柳等与人物院落的象征对应
4. **复原**：脂评本未分 17/18 回、程高本分回对路径描写的影响
