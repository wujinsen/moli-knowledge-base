# 红楼梦 · 知识图谱架构

## 节点类型

| 类型 | 集合 | 路由 |
|------|------|------|
| 人物 | `characters/` | `/honglou/c/{id}` |
| 地点 | `locations/` | `/honglou/l/{id}` |
| 名物 | medicines, dishes, costumes, customs | `/honglou/i/{id}` |
| 意象 | `imagery/` | `/honglou/imagery/{id}` |
| 回目 | `chapters/` | `/honglou/read/{edition}/{n}` |

## 边来源

1. **人物 relations** — 实体 frontmatter `relations[]` → `scripts/build_relations.py` → `红楼梦.relations.json`
2. **跨链** — `红楼梦.crosslinks.json`（occupant↔item、location↔item）
3. **意象互文** — 实体 `links[]` + `红楼梦.imagery-links.json` → `/honglou/shi`

## 回目 frontmatter（只改 YAML）

- `characters[]`、`locations[]`、`items[]`、`summary`
- `items[]` 仅实体 id；地点清洗见 `clean_chapter_items.py`

## 质量工具

| 脚本 | 作用 |
|------|------|
| `build_relations.py` | 重算人物图 |
| `trust_guard.py` | first_appear 原文抽检 |
| `lint_kb.py` | 孤儿/缺页/缺字段报告 |
| `_audit_summaries.py` | title-auto 梗概审计 |

## 版本

脂砚斋本（80）与程高本（120）正文只读；差异用回目 `edition` 字段与人物 `variants` 并列，`contradicts` 标记议题 id。
