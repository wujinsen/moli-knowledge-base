# schema: 关系受控词表

`relations[].type` 只能取以下值，写错会被构建期 Zod 校验拦下。

## 词表

```
亲属: 夫妻 父子 母子 兄弟 姐妹 祖孙 妯娌
社会: 主仆 师徒 师兄弟 同僚 朋友 结拜 君臣
情感: 情人 恋慕 仇敌 敌对
金瓶梅: 帮闲 贿赂 借贷 认干亲 庇护 资助 嫉妒 陷害
```

## 约定

- 双向关系（夫妻、兄弟、朋友、结拜、敌对…）只在一方写一条即可，
  `build_relations.py` 会统一去重。
- 有方向的关系（父子、母子、主仆、师徒、君臣）用 `role` 注明本人角色，
  如 `{ target: 唐僧, type: 师徒, role: 徒弟 }`。
- 新增词表项需同步更新 `src/content.config.ts` 的 `RELATION_TYPES`。
