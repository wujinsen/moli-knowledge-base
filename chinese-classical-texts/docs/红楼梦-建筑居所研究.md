> 记录日期：2026-06-14  
> 定位：`classical-novels` 空间实体样板，与 [`红楼梦-饮食纵切研究.md`](红楼梦-饮食纵切研究.md) 并列  
> 数据：`apps/classical-novels/src/content/locations/红楼梦/` · `src/data/红楼梦.locations.json`

## 一、方法论

以**建筑 / 居所 / 园林**为透镜，把分散在 120 回中的空间描写、匾额对联、居住分派、宴饮路径聚合成可检索、可图谱化的实体层。

一道建筑天然连接：**谁居（人物）+ 何匾（文学）+ 哪回（时间）+ 何情节（事件）+ 何陈设（物质文化）+ 上级空间（层级）**。

## 二、新增本体

见 [`schema/location.md`](../apps/classical-novels/schema/location.md)：

```
location 地点/建筑
  id / type / name / book
  category: 府邸|花园|院落|亭|楼|阁|榭|堂|馆|庵|祠|闸|幻境|市街|其他
  parent → 上级 id
  occupants[] → 人物 id
  plaque / couplet / features[] / furnishings[] / plants[]
  first_appear / appear_in[]
```

回目 `locations[]` 存 id 引用，详细字段在 location 页维护。
