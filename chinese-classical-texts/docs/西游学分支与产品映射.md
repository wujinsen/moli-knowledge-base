## 西游学是什么

**西游学**是围绕《西游记》的跨学科研究。与红学聚焦「家族兴衰 + 精密考证 + 悲剧纪实」不同，西游学面对的是一部**历经数百年民间演化**、融合神话、宗教、政治隐喻与市井幽默的奇书。

| 对比 | 红学 | 西游学 |
|------|------|--------|
| 研究对象 | 未竟稿贵族小说，写实色彩浓 | 世代累积型神魔小说，幻想色彩浓 |
| 学者姿态 | 拿放大镜还原「已坍塌的真实贵族大厦」 | 一层层剥开「全民族共同幻想了几百年的文化洋葱」 |
| 外层 | 文学人物与名物 | 狂欢化喜剧、单元剧叙事 |
| 中层 | 社会制度、礼教批判 | 政治讽刺、民间神话收编 |
| 内核 | 曹雪芹身世、版本探佚 | 儒释道「修心」、内丹心性论 |

本项目 `classical-novels` 对西游记应启用 **`bestiary` + `items` + GIS 地图** 等专属模块（见 `books/西游记.md` 的 `features`），而非套用红楼梦的四维图谱模板。

## 一、源流演变与作者考证（成书史）→ raw + 考证台

《西游记》非一人一时闭门造车，而是从玄奘《大唐西域记》→ 宋代《取经诗话》→ 元杂剧 → 明百回本的长链 **世代累积**。

| 研究方向 | 内容 | 项目映射 | 数据层 | 功能 ID |
|----------|------|----------|--------|---------|
| **文本演化** | 唐记 → 诗话 → 元杂剧 → 明小说 | 源流 **timeline** topic；各阶段文献并列索引 | `raw/daizhige/`（诗话、杂剧等）；`wiki/topics/西游记/` | H-10, B-15, T-06 |
| **作者公案** | 丘处机说 vs 吴承恩定调 vs 李春芳等异议 | **假说卡**（inference）多作者分支；禁止写死为事实 | `topic` 页 + `variants` | H-11, B-17 |
| **版本对勘** | 世德堂本 vs 清代评点本等 | 多 `edition` 并列；异文 `variants`；对勘 UI | `raw/editions/xiyouji/`（世德堂本 ✅） | H-10, H-11, B-16 |

**工程原则**（与红学版本学相同）：

- raw 只读；世德堂本为展示主本，其它刻本/文献并列不合并。
- 作者归属、丘处机说等**学术争议一律标诠释**，挂文献证据链。
- 殆知阁另有《取经诗话》《西游记补》《续西游记》等，可作源流链节点 catalog，按需 ingest。

## 二、宗教内涵与哲学寓意（三教合一 · 内丹学）→ topic + 象征图谱

表面佛教取经，内藏道教术语与儒家伦理；清代刘一明、悟元子等及现代宗教研究者视其为 **内丹修仙寓言**。

| 研究方向 | 内容 | 项目映射 | 功能 ID |
|----------|------|----------|---------|
| **全真内丹学** | 取经 = 炼丹；悟空 = 心/金，八戒 = 肾/木；八十一难 = 心魔 | `symbol` 实体；人物—五行—脏腑 **inference 边** | B-03, R-10, T-06 |
| **心性论 · 禅宗** | 「心猿意马」；大闹天宫 = 放心 → 斗战胜佛 = 收心 | `topic` 主题页；孙悟空 **状态演化**（B-13） | B-11~13, T-05, T-06 |
| **三教合一** | 佛/道/儒符号并置 | 门派标签 `faction`；神仙谱系 `deity` 子类 | B-03, B-07 |

**边属性**：内丹对应、象征解读 → `inference: true`，图谱虚线展示（与红楼「影射」边同机制）。

## 三、社会政治隐喻与世俗批判 → 背景论 + 天庭官僚图谱

借神魔外壳写明代社会骨相：天庭 = 皇权官僚；「有背景妖怪被接走、草根一棒打死」= 特权批判。

| 研究方向 | 内容 | 项目映射 | 功能 ID |
|----------|------|----------|---------|
| **天庭官僚体制** | 玉帝、天庭职官、拉帮结派、按资排辈 | `location`（天庭/地府）；`character` 职官字段；官僚 **关系子图** | R-05~06, B-11 |
| **背景论** | 坐骑/童子 vs 草根妖；被收 vs 被打死 | 妖怪 `背景` / `主人` 属性；`结局` 分类统计 | R-07, H-06, B-08 |
| **黑色幽默** | 讽刺、欺上瞒下 | `topic` 专题；回目 `tags` | T-01, wiki `topics/` |

**数据示例**（monster 扩展，规划）：

```yaml
# type: monster
背景: 有          # 有 | 无
主人: 观音菩萨
结局: 被收走
```

## 四、神话渊源与民俗学 → bestiary + 原型链 + GIS

《西游记》是神话素材库；民俗学追踪信仰遗迹与跨国原型。

| 研究方向 | 内容 | 项目映射 | 功能 ID |
|----------|------|----------|---------|
| **孙悟空起源** | 无支祁 vs 哈努曼 vs 混血说 | 妖怪/人物 `原型` 字段；**多假说 topic** | H-11, B-17, T-06 |
| **神仙谱系收编** | 二郎神、哪吒、龙王、城隍土地 | `character`/`deity` + `tags`；民间信仰 ←→ 小说 **收编边** | B-03, B-07, R-08 |
| **劫难地理** | 取经路线、真实地理 + 神话地理 | `location` 双层；**R-08 西游 GIS 地图** | R-08, B-02 |

**已落地模块**：`features: bestiary` → `/xiyouji/bestiary` 妖怪图鉴；`monster` schema 含 `洞府` `原型` `法宝` `收服者`。
