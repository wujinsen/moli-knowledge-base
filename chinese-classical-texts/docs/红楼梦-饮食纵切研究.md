## 一、方法论：纵切研究（Vertical Lens）

知识图谱最怕「为建图而建图」。选一条贯穿性主题，沿它把分散实体重新聚合，逼出隐藏结构——这就是纵切研究。

**饮食为何是绝佳透镜**：一道菜天然是高价值边——谁吃（人物）+ 在哪吃（地点）+ 哪一回（时间）+ 什么场合（事件）+ 多少钱（经济）+ 寒热属性（医理）。建一张饮食子图，等于顺手织出大半张大观园社会网。

## 二、能推演出什么（四层）

### 1. 人物地位 — 饮食是最诚实的阶层标尺

| 信号 | 原文证据 | 可量化指标 |
|------|----------|-----------|
| 工艺奢侈度 | 茄鲞「十来只鸡来配它」（刘姥姥惊叹） | 食材数 / 工序数 |
| 专属性 | 贾母专享「老君眉」、独享软烂菜 | 独享 vs 共享 |
| 城乡落差 | 刘姥姥的鸽子蛋、茄鲞反应 | 见识冲突值 |
| 主仆分野 | 主子吃席、丫鬟吃「剩的」「另装」 | 饮食流向（自上而下） |

→ 可算「饮食地位分」，与关系图中心性交叉验证：吃得好的人是否真有权。

### 2. 人物寿命 / 健康 — 饮食—体质—命运链（须分层）

- **事实层**：黛玉常年燕窝粥 + 人参养荣丸、饮食清淡量少；宝钗「冷香丸」；秦可卿病中饮食；贾母重食疗养生。
- **推论层（假说卡）**：饮食清简寡淡 ↔ 体弱多病 ↔ 早夭。标注「诠释，非定论」，挂全部原文出处，允许反方。

→ 功能：「饮食—病症—寿命关联面板」，选人物自动汇集其饮食/药膳/病症并按回目排时间线。

### 3. 社会饮食文化体系 — 明清贵族饮食全景

- **分类**：主食 / 汤羹 / 茶 / 酒 / 点心 / 药膳 / 节令食 / 贡品
- **节令轴**：元宵、螃蟹宴、中秋、寿宴——宴席是情节高潮节点，串成宴会时间线
- **南北 / 满汉**：曹家江南织造背景 → 江南风味 + 北方官府菜混搭
- **经济**：食材估价沿全书铺开 → 「饮食奢侈度曲线」，与贾府兴衰曲线叠合（食物由精致转简朴 = 败落隐喻）

### 4. 作者认知关系 — 饮食是曹雪芹的记忆考古

- 书中饮食真实可复现性远超同代小说，反映曹家「钟鸣鼎食」亲历记忆
- 标注：真实清代官府菜 / 艺术虚构 / 作者价值判断（借饮食写奢靡之戒）
- 版本异文：不同抄本对同一席描写差异，反映成书与修改痕迹

## 五、关键原则：事实与推论分层

学术可信度命门——原文写了什么（fact）与我们推断了什么（inference）必须分开存、分开显示：

- 事实边：`黛玉—吃—燕窝粥 @第45回`（可溯源，Trust Guard 校验）
- 推论卡：`饮食清简 ⇒ 体弱早夭`（标「诠释」，列全部支撑证据，允许反方）

同一份数据三种用法：读者看故事、学者看论证、师生看考点。
