## 金学是什么

**金学**是围绕《金瓶梅》的跨学科研究。若说《红楼梦》是没落贵族的挽歌、《西游记》是神魔修心之旅，《金瓶梅》则是**照出晚明市井百态与人性欲望的照妖镜**。

学者早已超越「情色描写」的刻板印象，将其视为蕴含**明代社会运转逻辑**的超级富矿——更像一群**戴着显微镜的明代社会人类学家**：测量白银纯度、听运河商埠叫卖声、解剖金钱至上、物欲横流的时代阵痛。

| 对比 | 红学 | 西游学 | 金学 |
|------|------|--------|------|
| 基调 | 悲剧纪实、贵族挽歌 | 浪漫神魔、修心寓言 | 世情写实、欲望显微镜 |
| 图谱重心 | 四大家族 + 大观园 + 名物 | 神魔 + 劫难 + GIS | **政商网络 + 物价 + 物质百科** |
| 版本复杂度 | 脂本 vs 程高 | 世德堂等 | **词话 / 崇祯 / 张竹坡** 三大系统 |
| 最大悬案 | 曹雪芹、后四十回 | 吴承恩（相对有主） | **兰陵笑笑生身份**（最大作者公案） |

本项目对金瓶梅应启用 **B-04 金瓶梅本体**（官职、货币、服饰、饮食、曲牌、药材），并优先建设 **物价可查询 + 版本对勘 + 纵切名物**（见 [`产品功能规划.md`](产品功能规划.md) B-07~08、H-06）。

## 一、作者之谜（兰陵笑笑生）→ 考证台 + 假说卡

《金瓶梅》署名「兰陵笑笑生」——性别不详、真实身份未定论，为**中国文学史最大作者悬案之一**。

| 研究方向 | 内容 | 项目映射 | 数据层 | 功能 ID |
|----------|------|----------|--------|---------|
| **身份考证** | 王世贞（为父报仇传说）、李开先、屠隆、汤显祖等几十种假说 | **假说卡**（inference）；多候选人 topic 树 | `topics/金瓶梅/作者公案/` | H-11, B-17 |
| **内证分析** | 方言、官场规矩、诗词水平反推作者 | topic + 引文锚点 | `wiki/` + chapter `source` | H-12, B-15 |
| **抄本与版本对勘** | 词话本（万历）、崇祯本、张竹坡评本 | 三版本 `edition` 并列；`variants` | `raw/daizhige/`、`raw/editions/jinpingmei/` | H-10, H-11, B-16 |

**工程原则**：

- 展示主本：当前 ingest 为殆知阁 **词话本**（`books/金瓶梅.md`）；崇祯本、张竹坡本 raw 并列，不合并。
- 所有作者归属假说标 **诠释**，禁止写入 character 为「事实作者」。
- 兰陵 = 山东临沂兰陵县，与 §四 方言研究交叉引用。

## 二、晚明社会经济史与商业文明 → 物价本体 + 政商图谱

中国第一部真正的**世情小说**，被历史学家当社会学调查报告读。

| 研究方向 | 内容 | 项目映射 | 功能 ID |
|----------|------|----------|---------|
| **西门庆发家史** | 破落户 → 高利贷 → 生药铺 → 包揽诉讼 → 认蔡京为干爹 → 理刑千户 | `character` 关系 + **时序图谱**（B-11~13） | R-05~06, B-11~12, H-02 |
| **政商勾结** | 权力寻租、干亲、送礼、诉讼包揽 | 边类型：`认干亲`、`贿赂`、`庇护` | B-03, H-04 |
| **货币与物价** | 银铜比价、买丫鬟市价、酒席成本；一条鞭法后白银货币化 | **`price` / `transaction` 实体**；数值型属性 | B-04, B-07, B-08, H-06 |
| **店铺与产业** | 生药铺、绒线铺、绸绒铺… | `location` / `business` 节点 | B-07 |

**复合查询示例**（B-08）：

> 金瓶梅 · 服饰 · 价值 > 10 两 · 出现在第 27 回前后

## 三、极端细致的物质文化与世俗生活 → 纵切百科图谱

若对金瓶梅做实体抽取，将得到**庞大无比的明代生活百科图谱**——本项目名物层的**第二标杆**（第一标杆为红楼饮食纵切）。

| 研究方向 | 内容 | 项目映射 | 纵切 / schema |
|----------|------|----------|---------------|
| **服饰** | 「大红刻丝段子对衿圆领」等材质、色、式 | `costume` + 数值/材质属性 | `schema/costume.md` |
| **饮食** | 猪头肉做法、各类酒、西门府宴饮 | `dish` / `banquet` | 可复用 [`红楼梦-饮食纵切研究.md`](红楼梦-饮食纵切研究.md) 范式 |
| **医药与性学** | 海量方剂、药材；春药、性具 — 晚明纵欲心理 | `medicine` + `topic`（学术诠释分层） | `schema/medicine.md` |
| **曲牌与娱乐** | 散曲、小调、戏剧；勾栏瓦舍 | **`qupai` / `performance` 实体**（待建） | B-04, T-08~10 |
| **礼仪民俗** | 婚丧、节庆、奴婢买卖 | `custom` | `schema/custom.md` |

**纵切文档（规划）**：`金瓶梅-物价货币研究.md`、`金瓶梅-服饰饮食研究.md`（可与红楼纵切并列模板）。

## 五、人性解剖与文学叙事革命 → 人物网 + 金红递进

打破非黑即白套路，书写普通人/小人物；女性群像与《红楼梦》的文学史承接是核心。

| 研究方向 | 内容 | 项目映射 | 功能 ID |
|----------|------|----------|---------|
| **复杂女性群像** | 潘金莲：妻妾制度下的焦虑、嫉妒、反叛 | `character` 深度页；避免标签化 `tags` | R-01~03, T-01 |
| **网状家族叙事** | 西门府内妻妾、奴婢、帮闲 | 关系图谱 + SNA | R-05~06, H-01~04 |
| **金红递进** | 无金无红；曹氏借鉴家族叙事、日常细节、主仆关系 | **跨书 topic**；对比页 | T-06, wiki `topics/` |
| **欲望与衰败弧** | 西门庆个人—家族—政治网崩塌 | 时序图谱关键帧（第 1/50/100 回） | B-14, B-13 |
