> 记录日期：2026-06-14

## 一、项目定位

**moli-knowledge-base** 是多领域 LLM Wiki 工作区（[Karpathy 模式](https://gist.github.com/karpathy/442a6bf555914893e9891c11519de94f)），演进目标见 **[`2026智能知识库演进路线.md`](../../docs/2026智能知识库演进路线.md)**（Agent · Graph RAG · 泛模态 · 知识找人 · 溯源）。

当前唯一落地领域：**chinese-classical-texts（中国古籍）**，采用三层架构：

```
raw（只读真相源）→ wiki（LLM 维护）→ apps（静态展示）
         ▲                    │
         └──── Trust Guard ───┘
```

其它领域（保险、教育、IT…）规划为**同级兄弟目录**，各自独立 raw/wiki/apps。

## 二、已落地功能（可运行）

### 2.1 章回小说前端 `apps/classical-novels`

| 功能 | 路由 / 命令 | 状态 |
|------|-------------|------|
| **选书首页** | `/` | ✅ |
| **单书目录** | `/{slug}`（honglou / jinpingmei / xiyouji） | ✅ |
| **阅读器** | `/{slug}/read/{chapter}` | ✅ 320 回；上一回/下一回；键盘 ←→ |
| **人物页** | `/{slug}/c/{id}` | ✅ |
| **关系图谱** | `/{slug}/graph` | ✅ **沉浸式探索器**（见 [`图谱可视化方案.md`](图谱可视化方案.md)） |
| **妖怪图鉴** | `/{slug}/bestiary` | ✅ 西游记 features 启用 |
| **取经路线 GIS** | `/{slug}/route` | ✅ 西游记双层地图（凡间 22 站 + 神话异界 5 处） |
| **全文搜索** | `/search` | ✅ Pagefind 静态索引；中文 zh-cn；按书/类型筛选 |

**技术栈**：Astro 6 · React 19 · Tailwind 4 · ECharts 6 · TypeScript · `astro-pagefind` v2

**常用命令**（在 `apps/classical-novels/` 下）：

```bash
npm run dev          # 开发（搜索读上次 build 的索引）
npm run build        # 构建 + 生成 Pagefind 索引
npm run preview      # 预览含搜索
npm run import       # 从 raw 导入回目
npm run relations    # 生成 relations.json
npm run guard        # Trust Guard 校验
```

**关系图谱（V1）**：深色全屏 `GraphLayout` · 节点发光 · 关系着色 · 搜索/阵营筛选 · 邻接高亮 · 侧栏详情 · 全屏 · 推论虚线边。详见 [`图谱可视化方案.md`](图谱可视化方案.md)。

- 构建期 Pagefind 索引，零后端

- 完整构建后约 **336+ 页**可检索（随 content 增长）
